Para executar o projeto:
1 - Abra a pasta cliente e execute o comando "npm install"
2 - Abra a pasta nodejs e execute o comando "npm install"
3 - abra o cmd e abra a pasta nodejs e execute o comando "nodemon index.js"
4 - abra a pasta cliente e execute o comando "ng serve"
5 - abra o projeto no navegador no endereco "http://localhost/4200"
